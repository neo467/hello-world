Set Up a MetaMask Wallet

The most widely used EVM wallet. Takes about 5 minutes.

💻 Desktop — Recommended 📱 Mobile
🌐
On Desktop
Go to metamask.io → Download → Install Chrome/Firefox extension → Click "Create a New Wallet"
📱
On Mobile
App Store / Google Play → Search "MetaMask" (fox icon 🦊) → Install → Tap "Create a New Wallet"
🔑

You'll receive a 12-word Secret Recovery Phrase. Write it down on paper and store it safely. If you lose these words, your wallet is gone forever.

2

Get USDC on Base Network

$LAMB is purchased with USDC on the Base network.

⭐
A. Withdraw from an Exchange — Easiest Method!
Have USDC on Binance / Bybit / Coinbase / OKX
→ Copy your MetaMask wallet address (0x...)
→ Exchange → "Withdraw" → Select USDC → Paste address
⚠️ Select "Base" as the network! (NOT ERC-20 / Ethereum!)
🌉
B. Use a Bridge
Already have USDC on another chain? Use bridge.base.org or Orbiter Finance to bridge it to Base.
🔵
C. Coinbase Users
Coinbase App → Withdraw USDC → Select Base network → Send to your MetaMask address
🦊
D. Swap inside MetaMask — Already have ETH/other tokens?
You can swap any token to USDC directly inside MetaMask — no exchange needed.
